# Getting Started
Install the dependencies and run the project
```
npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite
## About Twimba

Twimba is a Twitter-like web application developed with HTML, CSS, and JavaScript. It enables users to post tweets, like, retweet, and comment while showcasing JavaScript fundamentals such as DOM manipulation, event handling, and dynamic content rendering.
